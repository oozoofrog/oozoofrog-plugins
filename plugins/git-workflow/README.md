# git-workflow

GitHub 이슈 생성 + 브랜치 + Projects 보드 연동 자동화 플러그인.

## 스킬

### `/add-git-issue`

사용자의 요청을 구조화된 GitHub 이슈로 변환하고, 대응 브랜치를 생성합니다.

**주요 기능:**
- 코드베이스 조사 후 관련 파일/라인 번호를 포함한 상세 이슈 본문 작성
- 이슈 유형(버그/기능/디자인/리팩터링) 자동 판별 + 브랜치 패턴 적용
- `version.json` 기반 마일스톤 자동 연동
- GitHub Projects V2 보드 연동 (선택)

**사용 예:**
```
사용자: 로그인 화면에서 비밀번호 입력 시 키보드가 안 닫혀
→ 코드 조사 → 버그 이슈 생성 → fix/123-keyboard-dismiss 브랜치 생성
```

## 설치

oozoofrog-plugins 마켓플레이스를 통해 설치됩니다.
